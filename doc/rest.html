REST Interface Description

Conceptpower offers the following methods to retrieve information:

  • Retrieve a concept by its ID or URL
  • Search for a concept using a word and its POS
  • Retrieve a type by its ID or URL
  • Search for a concept
  • Get all synonyms for a word
